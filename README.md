# YT Silent Deep Video Creator

Automated video creation using Remotion.

## How to Generate a Video

1. Place your media in `public/mats/[video-name]/`.
2. Run the following command:

```powershell
npm run make -- [video-name]
```

Example:
```powershell
npm run make -- 01-tartary-submarine
```

The video will be rendered to the `out/` directory.
